# Secure Shell - Защищённая программная оболочка

## Описание

**Secure Shell** - это десктопное приложение на Python, представляющее собой полнофункциональную программную оболочку, имитирующую основные возможности операционной системы Windows с жёсткой системой разграничения доступа по направлениям деятельности и уровням секретности.

## Основные возможности

✅ **Авторизация и управление пользователями**
- Форма входа с проверкой учётных данных
- Создание и управление пользователями (для администратора)
- Хеширование паролей с солью (SHA-256)

✅ **Разграничение доступа**
- Три направления: Север, Запад, Юг
- Четыре уровня секретности: Несекретно, Для служебного пользования, Секретно, Совершенно секретно
- Иерархическая модель доступа

✅ **Графический интерфейс**
- Рабочий стол с иконками быстрого доступа
- Панель задач с кнопкой «Пуск» и системным треем
- Проводник (файловый менеджер) с фильтрацией по правам
- Меню «Пуск» с доступными приложениями

✅ **Встроенные приложения**
- Текстовый редактор (.txt, .rtf, .docx)
- Табличный редактор (.csv, .xlsx)
- Панель управления и администрирования

✅ **Защита данных**
- Контроль доступа на уровне БД и UI
- Логирование всех действий пользователей (аудит)
- Автоматическое наследование прав при создании файлов

✅ **Управление сеансами**
- Сохранение состояния рабочей среды
- Восстановление открытых окон и файлов при входе

## Требования к системе

- Python 3.10+

## Первые шаги

### 1. Учётные данные администратора (по умолчанию)

При первом запуске приложения автоматически создаётся администратор:
- **Логин:** `admin_admin`
- **Пароль:** `12345678`
- **Направление:** Север
- **Уровень:** Совершенно секретно
- **Статус:** Администратор

### 2. Создание пользователя

1. Авторизуйтесь с администраторским учётом
2. Откройте **Панель управления** → **Администрирование**
3. Нажмите **Добавить пользователя**
4. Заполните форму с указанием:
   - Логина (формат: `Фамилия_ИО`, например `Иванов_ИИ`)
   - Пароля (8 цифр или произвольный)
   - Направления (Север/Запад/Юг)
   - Уровня секретности
5. Нажмите **OK**

## Использование

### Проводник
- Просмотр только разрешённых папок и файлов
- Создание/удаление/переименование файлов и папок
- Копирование/перемещение в пределах разрешённых областей
- Просмотр свойств (направление, уровень секретности)

### Текстовый редактор
- Создание и редактирование документов
- Поддержка форматов: .txt, .rtf, .docx
- Базовое форматирование текста
- Автоматическая проверка прав при сохранении

### Табличный редактор
- Работа с таблицами (ячейки, строки, столбцы)
- Простые формулы и сортировка
- Импорт/экспорт .csv и .xlsx

## Безопасность

### Защита паролей
- Все пароли хранятся в виде SHA-256 хешей с солью
- Пароли никогда не хранятся в открытом виде

### Контроль доступа
- Двухуровневая проверка: UI-уровень (скрытие элементов) и бэкенд (проверка прав)
- Все операции с файлами проверяют права доступа
- Направление и уровень секретности проверяются для каждой операции

### Логирование
- Все действия пользователей записываются в журнал аудита
- Логируется: вход/выход, открытие файлов, создание/удаление, ошибки доступа
- Файл логов: `data/audit.log`
